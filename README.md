# GlideTP - Minecraft 텔레포트 모드

간편한 텔레포트 기능을 제공하는 모드입니다.

## 주요 기능

### 기본 텔레포트
- **좌표 이동**: `/tp <x> <y> <z>` - 특정 좌표로 이동
- **플레이어 이동**: `/tpto <player>` - 다른 플레이어에게 이동

### 홈 시스템
- `/sethome [name]` - 현재 위치를 홈으로 설정 (기본: "home")
- `/home [name]` - 설정한 홈으로 이동
- `/delhome <name>` - 홈 삭제
- `/homes` - 내 홈 목록 보기

**특징:**
- 플레이어당 여러 개의 홈 설정 가능
- 이름을 지정하여 관리 (예: 집, 농장, 광산 등)
- 차원(Dimension) 정보 포함

### 워프 시스템
- `/setwarp <name>` - 공개 워프 포인트 설정 (관리자 전용)
- `/warp <name>` - 워프 포인트로 이동
- `/delwarp <name>` - 워프 삭제 (관리자 전용)
- `/warps` - 모든 워프 목록 보기

**특징:**
- 서버 전체가 공유하는 공개 이동 지점
- 관리자만 설정/삭제 가능
- 모든 플레이어가 사용 가능

### 도움말
- `/tphelp` - 모든 명령어 도움말 보기

## 설치 방법

1. **Minecraft 1.20.1** 설치
2. **Forge 47.3.0** 이상 설치
3. 빌드된 JAR 파일을 `mods` 폴더에 복사
4. 게임 실행

## 빌드 방법

```bash
# Windows
.\gradlew.bat build

# Linux/Mac
./gradlew build
```

빌드된 파일: `build/libs/GlideTP-1.20.1-1.0.1.jar`

## 사용 예시

### 홈 설정 및 사용
```
/sethome          # 기본 홈 설정
/sethome 농장     # "농장"이라는 이름의 홈 설정
/home             # 기본 홈으로 이동
/home 농장        # "농장" 홈으로 이동
/homes            # 내 홈 목록 확인
/delhome 농장     # "농장" 홈 삭제
```

### 워프 사용 (관리자)
```
/setwarp 스폰     # "스폰" 워프 설정
/setwarp 상점     # "상점" 워프 설정
/warps            # 워프 목록 확인
```

### 워프 사용 (일반 유저)
```
/warp 스폰        # 스폰으로 이동
/warp 상점        # 상점으로 이동
```

### 기본 텔레포트
```
/tp 100 64 200    # 좌표로 이동
/tpto Steve       # Steve에게 이동
```

### 도움말
```
/tphelp           # 모든 명령어 보기
```

## 데이터 저장

모든 데이터는 서버 폴더의 `tpmod_data/` 디렉토리에 JSON 형식으로 저장됩니다:
- `homes.json` - 플레이어별 홈 데이터
- `warps.json` - 공개 워프 데이터

## 권한

- **일반 유저**: 홈 관리, 워프 사용
- **관리자 (OP)**: 모든 기능 + 워프 설정/삭제, 좌표 텔레포트
